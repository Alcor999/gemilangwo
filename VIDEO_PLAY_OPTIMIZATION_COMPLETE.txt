HOW TO COMPRESS VIDEOS
================================================================================

OPTION 1: HANDBRAKE (Desktop App - Free)
  1. Download: handbrake.fr
  2. Open video file
  3. Choose "Very Fast" preset
  4. Set resolution to 720p
  5. Export MP4
  6. Result: ~100MB for 5 min video

OPTION 2: FFMPEG (Command Line)
  For 720p (fast):
  ffmpeg -i input.mp4 -vf scale=1280:720 -c:v libx264 -crf 28 -b:v 4M output.mp4
  
  For 1080p (balanced):
  ffmpeg -i input.mp4 -vf scale=1920:1080 -c:v libx264 -crf 23 -b:v 5M output.mp4

OPTION 3: ONLINE TOOLS
  - TinyWow (tinywow.com/video/compress)
  - CloudConvert (cloudconvert.com)
  - VideoSmaller (videosmaller.com)

OPTION 4: YOUTUBE UPLOAD
  - Upload to YouTube (unlisted/private)
  - Use YouTube URL in system
  - No server storage needed!
  - Best for large videos
